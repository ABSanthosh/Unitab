# Unitab

A simple new tab page for Chrome.


## Features
- [ ] Tab management
  - [ ] Tab ordering with ctrl+<number>
  - [ ] Horizontal tabs
- [ ] notes
- [ ] todo list
- [ ] weather
- [ ] clock
- [ ] workspaces
- [ ] bookmarks
- [x] Cats!

- [ ] iframe: [Driftmas](https://slowroads.io/driftmas/)
- [ ] Neko in all the pages
